# Synapic.NET Sidecar

This repository contains the sidecar application for Synapic.NET.

## Overview

The Synapic.NET Sidecar is a companion application designed to provide additional functionality and integration capabilities for the main Synapic.NET platform. It handles tasks such as:

-   **Model Management:** Browsing, downloading, and managing AI models (Hugging Face, Ollama, etc.).
-   **Inference:** Running local inference using various backends.
-   **Integration:** Connecting with external services like Daminion and social media platforms.

## Key Features

-   **Model Browser:** A GUI for searching and filtering compatible AI models.
-   **Local Inference Engine:** Support for running models locally to ensure data privacy and reduce latency.
-   **Daminion Integration:** Tools for tagging and organizing assets within Daminion catalogs.
